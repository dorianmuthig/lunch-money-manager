Lunch Money Manager
===

## This is unfinished non-working software 

This software is incapable of accomplishing any useful task. It is built with broken dependencies, at least one of which (SQLite) has an unpatched (at least through the EF Core dependency tree) security vulnerability as of writing.
I have no plans of finishing this software, and it merely serves as a demo for the person(s) who requested it.
