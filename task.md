Software developer – Coding Task
===
## General
We are very happy that you decided to take on this coding task.
Please use Visual Studio, .NET and C# in your solution. Make sure you think about **architecture, patterns, testability, extensibility and code style**. Ideally you also write tests.
Feel free to extend the application to meet your real-life needs. Please document how others can use the application.
## Problem Description
Your software development team wants to go to lunch together. Unfortunately, your colleague Daniel forgot his lunch money. Because you are super kind, you offer Daniel to lend him the money for his lunch. Because you are a structured person you want to keep track to whom you gave out money in case this happens more often.
## Task
Write a small application (console application is fine) which can take an input to store the name of the colleague and the amount of money in a file. If you think it makes sense to store more data,
feel free to extend your data.
After you inserted some loans of your colleagues, you want to be able to list all of them to get an overview.
Extend the application to meet your real-life needs (keep it simple and small but add real value).
## Duration
The task should not take longer than 3 hours.
## Delivery
Please send us the source code as zipped Visual Studio Solution. The documentation should be in a word document. Please format the results in PDF and send us a zip file.
Alternatively, you can also send us a OneDrive folder that we have access to.
